# ERP Sistemas - Laseclat

## Descripción

ERP Sistemas es un proyecto interno desarrollado para apoyar la administración y operación del área de Sistemas de Laseclat.  
El objetivo principal es centralizar herramientas, interfaces y módulos relacionados con la gestión tecnológica de la empresa.
